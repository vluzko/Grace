pub struct PosStr<'a> {
    pub offset: usize,
    pub line: u32,
    pub column: u32,
    pub slice: &'a [u8],
}

Fields§

§offset: usize

The offset represents the position of the slice relatively to the input of the parser. It starts at offset 0.

§line: u32

The line number of the slice relatively to the input of the parser. It starts at line 1.

§column: u32

The column number of the slice relatively to the input of the parser. It starts at column 1.

§slice: &'a [u8]

The slice that is spanned.
